# AGENTS.md

## 目的

エージェント共通の作業方針を定義する。

## 基本方針

- 会話は日本語で行う。
- コード内コメントは日本語、エラーメッセージは英語で記載する。
- コミットメッセージは Conventional Commits に従い、説明は日本語とする。
- 日本語と英数字の間には半角スペースを挿入する。

## 判断記録のルール

- 判断内容、代替案、採用理由、前提条件、不確実性を明示する。
- 仮定を事実のように扱わない。

## 開発手順（概要）

1. プロジェクトの理解（Maven リポジトリ構造と README 生成ツール）
2. 依存関係のインストール (`pnpm install`)
3. 変更の実装
4. テストと Lint/Format の実行 (`pnpm lint`, `pnpm start`)

## セキュリティ / 機密情報

- 認証情報や API キーを Git にコミットしない。
- ログに機密情報を出力しない。

## リポジトリ固有

- ルートディレクトリの Maven アーティファクトディレクトリ (`com/` など) の整合性を維持する。
- README の更新は `.github/generate-readme` ツールを使用して行う。
